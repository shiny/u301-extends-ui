## Introduction
You can add (copy) the target component to your shadcn UI project by running the following command:

**bun**
```bash
bunx shadcn add https://ui.tealight.uk/r/icon
```

**npm**
```bash
npx shadcn add https://ui.tealight.uk/r/icon
```
